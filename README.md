# Go Chat Server

### A high-performance instant messaging backend server supporting private/group chats, friend management, file transfers, and more.

## Project Status

This project was my first attempt at building a complete system with the help of AI.

At the time, it served as an exploration of what AI-assisted development could achieve in a relatively complex backend project. It helped me go from ideas to implementation much faster than I could have on my own.

Now, I feel that this project has fulfilled its purpose. I’ve decided to officially archive it and draw a line under this phase of experimentation.

The repository will remain available for reference and learning, but it is no longer actively maintained.

## Features

✅ User system (registration/login/profile management)
✅ Friend management (add/remove/block)
✅ Real-time private messaging (text/files)
✅ Group chat system (creation/management)
✅ Message operations (recall/edit/read receipts)
✅ SHA1-based file deduplication (supports files up to 128MB)
✅ JWT authentication (16-day validity with refresh mechanism)
